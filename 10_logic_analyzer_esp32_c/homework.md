## Питання для аналізу

2. У яких випадках мікроконтролер «пропускає» частину брязкоту, а коли навпаки — «перерахує зайве»?

Не зафіксовано

3. Яка середня тривалість брязкоту для вашої кнопки?


- 40-200 мікросекунд
- залежить від натискання
- не зафіксоано при натисканні кнопки, брязкіт тільки при відпусканні

4. Який мінімальний debounce (затримка / логіка фільтрації) був би достатній?


- більше 503 мікросекунд (найдовший зафіксований брязкіт)
