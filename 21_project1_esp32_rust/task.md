Розумний нічник
Пристрій повинен автоматично змінювати яскравість світлодіода залежно від рівня освітленості в приміщенні. Чим темніше навколо - тим яскравіше світить LED.

+ ​1. Рівень обробки сигналів (ADC)
+ ​Дискретизація: Частота зчитування даних з фоторезистора не менше 50 Гц (кожні 20 мс).
+ ​Цифрова фільтрація: Обов’язкова імплементація фільтра (Moving Average або EMA) для усунення високочастотного шуму ADC.
+ ​Калібрування: Програмна корекція нелінійності ADC ESP32-S3 (використання вбудованих характеристик esp_adc_cal).
+ ​Нормалізація: Мапування (Map) фільтрованих значень ADC (0...4095) у діапазон PWM (0...255 або 0...1023).

+/- ​2. Рівень керування (PWM/LEDC)
+ ​Конфігурація: Налаштування частоти PWM на рівні 5 кГц для відсутності видимого мерехтіння.
+ ​Плавність: Реалізація софтварного "fading" (плавного переходу) при різкій зміні освітленості, щоб уникнути стрибків яскравості.
- ​Лінеаризація: (Бажано) Застосування гамма-корекції для LED, оскільки людське око сприймає яскравість нелінійно.
+ Калібрування: Додати функцію автоматичного визначення мін/макс рівня світла при запуску.

+/- ​3. Логіка та стабільність
- ​Гістерезис: Впровадження мертвої зони (deadzone) для запобігання осциляціям ("тремтінню" світла) на межі спрацювання.
- ​Обробка помилок: Валідація значень сенсора (обробка сценаріїв обриву або короткого замикання фоторезистора).
+ ​Діагностика: Вивід значень (Raw ADC, Filtered, PWM %) у Serial Port для налагодження.
+ ​4. Архітектура
​+ Non-blocking code: Використання millis() замість delay() для забезпечення чутливості системи.

Основні вимоги, це фільтр ADC та non-blocking, решта по бажанню, ще можете погратися з ledc, добавити потенціометр для регулювання якогось з параметрів (максимальна яскравість, наприклад)
